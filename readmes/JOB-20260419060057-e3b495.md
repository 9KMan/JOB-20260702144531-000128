# Multi-Agent AI Technical Oversight Service

Production-grade technical oversight API for multi-agent AI systems. This service provides code review, architecture analysis, scaling assessment, and schema review capabilities powered by Anthropic's Claude.

## Features

- **Code Review**: Automated analysis of code with AI-powered insights
- **Architecture Review**: Deep-dive analysis of system design and schema
- **Scaling Assessment**: Metrics-based analysis for capacity planning
- **Schema Review**: PostgreSQL schema evaluation and recommendations
- **Comment Threading**: Line-specific code comments with resolution tracking
- **Recommendation Tracking**: Priority-ranked improvement recommendations

## Tech Stack

- **Backend**: Python 3.11, FastAPI, SQLAlchemy
- **Database**: PostgreSQL 15
- **AI**: Anthropic Claude API
- **Container**: Docker, Docker Compose
- **Testing**: pytest, pytest-cov

## Quick Start

### Prerequisites

- Python 3.11+
- PostgreSQL 15+ (or Docker)
- Anthropic API key

### Local Development

```bash
# Clone the repository
git clone https://github.com/9KMan/JOB-20260419060057-e3b495.git
cd JOB-20260419060057-e3b495

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Set environment variables
cp .env.example .env
# Edit .env with your DATABASE_URL and ANTHROPIC_API_KEY

# Run the application
uvicorn app.main:app --reload
```

### Docker Deployment

```bash
# Build and start services
docker-compose up -d

# View logs
docker-compose logs -f app

# Stop services
docker-compose down
```

## API Endpoints

### Reviews

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/reviews/` | Create a new review |
| GET | `/api/reviews/` | List all reviews |
| GET | `/api/reviews/{id}` | Get review by ID |
| PATCH | `/api/reviews/{id}/status` | Update review status |
| POST | `/api/reviews/{id}/comments` | Add comment to review |

### AI Analysis

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/ai/analyze-code` | Analyze code with AI |
| POST | `/api/ai/review-architecture` | Review architecture |
| POST | `/api/ai/assess-scaling` | Assess scaling needs |

### Health

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/` | Root endpoint |
| GET | `/health` | Health check |

## Review Types

- `code_review`: General code quality analysis
- `architecture`: System design and structure
- `scaling`: Capacity and performance analysis
- `schema`: Database schema evaluation
- `security`: Security vulnerability assessment

## Review Statuses

- `pending`: Awaiting review
- `in_progress`: Review underway
- `completed`: Review finished
- `approved`: Changes approved
- `rejected`: Changes require revision

## Running Tests

```bash
# Run all tests
pytest

# Run with coverage
pytest --cov=app --cov-report=html

# Run specific test file
pytest tests/test_api.py -v
```
