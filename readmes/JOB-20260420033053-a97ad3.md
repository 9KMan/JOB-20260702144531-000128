# DPIC - Real Estate Certification & Data Platform

A comprehensive platform for ingesting, processing, and certifying real estate data including rent rolls, ownership records, and entity information.

## Overview

DPIC provides a 3-level certification system:
- **Level 1: Title Integrity** - Validates property ownership and title history
- **Level 2: SPV/Entity Structural Integrity** - Verifies entity structures and authority
- **Level 3: Full Certification** - Complete validation and certification

## Tech Stack

- **Backend**: Python/FastAPI, PostgreSQL, SQLAlchemy
- **Frontend**: React, TypeScript, TailwindCSS
- **Database**: PostgreSQL with full-text search
- **Container**: Docker, Docker Compose

## Quick Start

### Backend

```bash
cd backend
cp env.example .env
docker-compose up -d
pip install -r requirements.txt
alembic upgrade head
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
