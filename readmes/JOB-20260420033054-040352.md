# ScaleFlow AI Automation Module

Production-ready AI automation module for ScaleFlow agency - a 14-day sprint deliverable.

## Tech Stack

- **Frontend**: Next.js 15, TypeScript, Tailwind CSS, Radix UI
- **Backend**: Next.js API Routes, Prisma ORM
- **Database**: PostgreSQL
- **AI**: OpenAI GPT-4, Google Gemini Pro
- **Automation**: n8n workflow engine
- **Multi-tenant**: SaaS with tenant isolation

## Features

- Multi-tenant workflow management
- AI-powered automation generation
- Integration with OpenAI/Gemini APIs
- Google Workspace API integrations
- Real-time workflow monitoring
- RESTful API with Next.js

## Quick Start

```bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env.local

# Generate Prisma client
npm run db:generate

# Run database migrations
npm run db:push

# Start development server
npm run dev
```

## Docker

```bash
# Build and run with Docker Compose
docker-compose up -d
```
