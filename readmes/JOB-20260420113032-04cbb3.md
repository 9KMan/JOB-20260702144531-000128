# MDM Platform v3.0

Master Data Management platform migration from v2 to v3 with new UI screens, PostgreSQL database, React frontend, and Node.js/GraphQL backend.

## Tech Stack

- **Frontend**: React 18, Apollo Client, React Router
- **Backend**: Node.js, Express, Apollo Server, GraphQL
- **Database**: PostgreSQL 16 with UUID, JSONB, pg_trgm
- **API**: GraphQL

## 8 Logical Domains

1. **Authorization** - Role-based access control (RBAC) with per-page permissions
2. **Source System Catalog** - Batch ingestion and lineage tracking
3. **Golden Records** - Record matching with full match provenance
4. **Relationships** - Households, organizations, identity documents
5. **Matching Engine** - Configurable thresholds and action rules
6. **Workflow** - Suspect/steward review workflow engine
7. **Audit Trail** - Immutable, field-level change history
8. **Reporting & AI** - Dashboard KPIs and AI search

## Quick Start

```bash
# Start all services
docker-compose up -d

# Or run locally
cd backend && npm install && npm start
cd frontend && npm install && npm run dev
```

## Database

Schema is in `database/schema.sql` with all 8 domain tables, triggers for audit logging, and proper indexes.

## API Endpoints

- GraphQL: `http://localhost:4000/graphql`
- Auth: `POST http://localhost:4000/auth/login`
- Health: `GET http://localhost:4000/health`

## Environment Variables

- `DATABASE_URL`: PostgreSQL connection string
- `JWT_SECRET`: Secret for JWT token signing
- `PORT`: Server port (default 4000)
