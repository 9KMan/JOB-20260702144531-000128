# Out of Scope

The following items are **not included** in this implementation:

- **Looker Studio dashboard** – Reporting uses static notebooks; no Looker Studio integration
- **GCP account setup** – No GCP project, service accounts, or IAM configuration
- **Scraping auth** – Authenticated scraping endpoints are not implemented
- **Real-time streaming** – No WebSocket or server-sent events for live audio streaming
- **Multi-tenancy** – Single-tenant only; no tenant isolation or per-org data partitioning
- **Mobile app** – No iOS/Android client; backend API only
- **Custom ML training** – Pretrained models used as-is; no fine-tuning or custom training pipeline
